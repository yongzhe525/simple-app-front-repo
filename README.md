# eks-cicd-dr_service-peccy-web

Peccy service web server.

## CLI Commands

```
# Local run
$ yarn start

# Unit test
$ CI=true yarn test

# Build
$ yarn build
```

## Main Packages

* Framework : React with webhook 
* OS environments : cra-envs
* API call : axios
